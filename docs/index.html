RBD (Base de datos relacionales)
  1. Entidades y atributos
    • Cualquier objeto del mundo real puede ser una entidad, se las coloca en plural porque representan a un conjunto de objetos. Se las encierra en un rectángulo
      • Ej: autos
    • Los atributos son características de las entidades
      • Ej: volante de un auto
    • Atributo multi-valuado
      • Ej: llantas de un auto, por que son varias va encerrada dos veces en el dibujo
    • Atributo compuesto
      • Ej: El motor de un auto, este a su ves tiene sus propios atributos
    • Atributo clave
      • Nos ayuda a identificar de manera única a la entidad, va subrayado

Instalar MySQL

Primero hay que actualizar el índice de paquetes en la pc si no se ha hecho recientemente.

                
sudo apt update

Luego se instala el paquete mysql-server:

                    
sudo apt install mysql-server

Comandos básicos

Los comandos no son case sensitive

La declaración CREATE DATABASE es usada para crear una nueva base de datos SQL.

                    
CREATE DATABASE databasename;

La declaración CREATE TEABLE es usada para crear una nueva tabla en una base de datos.


CREATE TABLE table_name (
    column1 datatype,
    column2 datatype,
    column3 datatype,
   ....
);

La declaración show databases; es usada para mostrar las bases de datos disponibles.


show databases;

La declaración show tables; permite ver los nombres de las tablas.


show tables;

La declaración use databasename; es usada para posicionarnos sobre la base de datos de la que queremos ejecutar acciones.


use databasename;

La declaración show tables from databasename; permite ver las tablas de una base de datos sin que nos posicionemos previamente.


show tables from databasename;

La declaración SELECT es usada para seleccionar columnas de una tabla.


SELECT column FROM table_name;
